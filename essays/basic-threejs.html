Creating a basic render in Three.js

To create a basic render in three.js, there are four requirements.

  1. Need a scene
  2. Some objects
  3. A camera
  4. A renderer

The scene can be thought of as a movie scene, with actors and props being the objects, and the cameras being, obviously, the cameras. Just like on a movie set, multiple cameras are possible, but usually there is no need given that a single camera can be moved almost instantaneously.

Three.js, being a library, has to be imported in order to be used. This can be done in several ways: by using a script, by using modules, or by using some sort of bundler like Webpack. If using a script, then download three.js from GitHub, and link the minified file in your HTML; you should have something like the following: 

<canvas id="webgl"></canvas>
<script src="./three.min.js"></script>
<script src="./script.js"></script>

The order of the scripts is important; the "three.min.js" file will be used in "script.js", therefore has to be linked first.

A scene is created using the following code:

const scene = new THREE.Scene();

To make a visible object, need to create a mesh, which needs geometry and material.

const geometry = new THREE.BoxGeometry(1, 1, 1);
const material = new THREE.MeshBasicMaterial({color: 0xff4a1a});
const cube_mesh = new THREE.Mesh(geometry, material);
scene.add(cube_mesh);
